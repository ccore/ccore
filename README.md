ccore
=====

A development library for games which makes it easier to use low level system functionality while maintaining cross-platform useabillity.

### Minimum requirements for compiling ###
**Linux:**
- scons (http://scons.org)
- libx11-dev
- libxrandr-dev

**Windows:**
- Visual Studio
